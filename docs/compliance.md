# Responsible data use

This repository contains original analytics code and synthetic demonstration
data. It does not include platform login state, cookies, account credentials,
raw user profiles, profile URLs, avatars or private information.

MediaCrawler may be used separately as a learning-only collection tool. Its
repository uses a Non-Commercial Learning License rather than MIT or Apache.
Do not copy its source into this repository. Review its current license and the
target platform's rules before every collection run.

Project rules:

1. Collect only information visible through an authorized normal user flow.
2. Do not bypass access controls, CAPTCHA, rate limits or technical safeguards.
3. Keep volume small, throttle requests and stop when the platform signals refusal.
4. Do not collect minors' data, private messages or sensitive personal information.
5. Hash post identifiers and remove account-level identifiers before analysis.
6. Publish only synthetic data or privacy-safe aggregates.
7. Do not redistribute copyrighted post text, images, video or comments.
8. Delete local raw data after the stated research retention period.

The portfolio should emphasize schema design, data quality, analysis and
communication—not circumvention or crawling scale.
