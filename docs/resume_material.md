# Resume material

## Chinese project title

BeautyPulse：小红书与抖音多国美妆内容洞察

## Resume bullets

- 围绕韩妆、泰妆、日妆、国际大牌及不同价格带国货美妆，设计跨平台内容数据模型，完成品牌标准化、互动指标清洗、品类识别及隐私脱敏流程。
- 使用 Python、Pandas 与规则型中文文本分析提取消费者卖点和痛点，并按平台与品类构建互动分位数，降低不同平台量级差异对比较结果的影响。
- 搭建 Streamlit 交互式看板，支持按平台、市场分组和产品品类筛选；通过单元测试验证计数解析、文本清洗和特征计算逻辑。

## Interview talking points

- Why medians are more robust than means for long-tailed engagement data.
- Why category control is needed when comparing brands across countries.
- Why collection time must be stored with cumulative engagement counts.
- Why the public repository contains synthetic data rather than raw social content.
- How a heuristic sentiment model differs from a supervised classifier.

## Honest scope statement

The project is a descriptive portfolio study. It does not claim that search
results represent all consumers, and it does not infer causal effects from
observational engagement data.
